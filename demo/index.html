Fullscreen box

A very used feature. A box always fullscreen for every devices and on resizing. Like the box on top this page.

<div class="fsbox banner"> <img class="bg" src="..." /> <h1 class="middle center">Some text</h1> </div>

The .img.bg class make image on background always on best view

The .middle class make text on.. middle ;)

Fix on scroll

Make an element fixed on the page when scroll over specified value in .data-c-fixto.
Right menu is an example

<div data-c-fixto="300"> .. </div> <div data-c-fixto="fullscreen"> .. </div>

With fullscreen value, take the window height.
